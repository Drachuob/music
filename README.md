# Music

Music script

```bash
Usage: ./script.sh [play|download|help]
Download and play music.

Options:
	play (default: play sounds)
		random: Play sounds randomly

	download (default: youtube)
		youtube: (default: mp3)
			mp3: Download to mp3 file format

Examples:
	./script.sh play
	./script.sh play <FILE>
	./script.sh play random

	./script.sh download <URL>
	./script.sh download youtube mp3 <URL>
```
